Four Stroke Cylinder

Designed a Four Stroke cylinder engine with the help of solidworks. Basically SOLIDWORKS drawings are more than simple illustrations. A four-stroke engine is an internal combustion engine in which the piston completes four separate strokes while turning the crankshaft. A stroke refers to the full travel of the piston along the cylinder, in either direction

Designed a 6 Jaw Hydraulic Chuck by using solidworks. Six jaw swivel grip chucks are a tool used in machines like lathes, milling, and many other machining capabilities. Twelve points of contact with jaws that swivel up to six degrees to compensate for part irregularities and out of roundness. Pullback feature provides maximum rigidity and length positioning.

Obstacle Avoiding Robot is an intelligent device that can automatically sense the obstacle in front of it and avoid them by turning itself in another direction. Maze Solving Bot

This project is an implementation of an autonomous robot that follows a derivative of the Wall Follower (Left-Hand) algorithm using appropriate sensors. The robot acts according to the code written in Arduino, which directs it to explore the maze containing random turns.

Fabrication

CNC Pen Plotter

The construction of the Project is based on computer numerical control machine where the goal was to make the simplest CNC machine with minimum parts possible. It uses 3 NEMA 17 stepper motors for the X, Y and the Z axis motion, and a small servo for the gripper. The brain of this CNC Pen plotter machine is an Arduino UNO board in combination with a CNC shield and three A4988 stepper drivers.
